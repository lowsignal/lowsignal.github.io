Framework CSS Terbaik

Ibrahim Al Anshor On 10 August 2019
  • CSS

Framework CSS merupakan sebuah pustaka/library yang berisi kode-kode css siap pakai untuk membuat halaman web. Tujuan penggunaan framework adalah untuk memudahkan dan juga pengerjaan sebuah web.

Bayangkan saja jika kita hendak membuat web dan harus membuat css satu persatu setiap projek, melelahkan bukan? Nah disitulah gunanya framework.

Sekarang sudah banyak framework siap pakai yang sudah tersedia, anda tinggal memilihnya saja.

Contohnya saja bootstrap. Bootstrap sudah digunakan untuk membuat berbagai web.

Atau bulma yang saya gunakan untuk membuat blog ini.

Langsung saja berikut Framework CSS Terbaik

Bootstrap

Bootstrap merupakan framework css berbasis flexbox. Boostrap mengutamakan responsive, jadi web anda akan terlihat bagus di platform manapun.

Bootstrap juga memiliki komonen yang lengkap, mulai dari alert, modal, dll. Selain komponen pallete warna pada bootstrap juga banyak serta bagus.

Boostrap sekarang berada di versi 4, anda dapat langsung menggunakanya dengan mendownload di situs resminya.

Bulma

Bulma merupakan framework yang sedang naik daun akhir-akhir ini. Bahkan blog ini juga menggunakan framework tersebut.

Salah satu keunggulan dari bulma adalah no javascript, jadi anda bisa menggunakan framework bulma tanpa menggunakan javascript.

Selain itu bulma juga sangat mudah digunakan karena berbasis flebox, komponennya lengkap.

Kustomisasi bulma juga sangat mudah, bulma juga menggunakan modular, artinya anda dapat menggunakan komponen bula yang hanya dibutuhkan saja.

Foundation

Jika anda ingin framework dengan fitur yang banyak, gunakanlah Foundation. Foundation juga dapat digunakan untuk membuat semua jenis web, mulai dari blog, dll.

Foundation juga dapat digunakain untuk HTML E-Mail.

Materialize

Materialize merupakan framework buatan google yang digunakan untuk membuat web dengan tampilan material.

Material merupakan style desain google yang kini populer dalam hal desain.

Dengan materialize kalian dapat membuat web dengan desain material.

Semantic UI

Framework ini memiliki komponen komponen yang elegan. Cocok untuk web dengan desain elegan.

Semantic UI juga memiliki nama-nama class yang mudah digunakan, sehingga pembuatan web akan jauh lebih mudah.

Tailwind

Tailwind merupakan framework yang sangat mudah dikustomisasikan. Cocok bagi kalian yang suka kustom-kustom.

Komponen-komponennya juga bagus dan keren.

Kesimpulan

Apapun frameworknya saya lebih milih bulma karena saya sudah mencobanya.
