Atribut Target

Dalam link kita juga dapat menyertakan atribut link. Dengan atribut target, kita dapat mengatur dimana link tersebut akan terbuka, di tab baru, window baru, dll. Target memiliki beberapa value, yaitu.

  • __blank - Membuka di tab baru
  • __self - Membuka di tab sama
  • __parent - Membuka di frame parent
  • __top - Membuka di tab yang sama secara full +
