Belajar HTML - Membuat Heading Pada HTML

Ibrahim Al Anshor On 09 August 2019
  • HTML

Heading merupakan tag html yang berguna untuk menampilkan judul pada halaman web. Heading dibuat dengan tag <h1> hingga <h6>

Syntax Dasar

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Penggunaan heading sangat dianjurkan karena disukai mesin pencari. Selain itu pengunjung juga menjadi lebih mudah dalam membaca halaman.

Horizontal Rules

Horizontal rules merupakan tag html yang menampilkan sebuah garis horizontal. Biasanya tag ini digunakan untuk membuat batas pada konten atas dan bawah.

Horizontal Rules dapat digunakan dengan syntax berikut.

Kesimpulan

  • <h1> - <h6> - Untuk membuat heading
  • <hr> - Membuat horizontal rules
